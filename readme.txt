Display your products' prices on your WooCommerce Category Archive pages.

== Description ==

This plugin allows you to **display your products' prices on WooCommerce Category Archive pages**. Works with **Regular price**, **Sale prices** and **lowest to highest price**. Also includes **Schema.org formated micro data**.

Prices are saved as metas in WooCommerce term_meta custom table.

== Installation ==

= Plugin installation =
1. Download this plugin and add it to '/wp-content/plugins/'
1. Activate it from WordPress &raquo; Plugins admin screen

= Plugin usage =
1. Create products and give them categories
1. Go to WooCommerce Settings &raquo; Products
1. Select 'show subcategory' instead of 'show all products'
1. Have a look at you shop ;)
